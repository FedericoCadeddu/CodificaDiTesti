#Progetto di Codifica di Testi

Progetto per il corso di codifica di testi del professore Angelo Mario Del Grosso. 

La codifica riguarda le pagine 7 e 8 del manoscritto "Diario di un partigiano ebreo".

Per la validazione del file **codxml.xml** ho utilizzato **Xerces** da prompt.

Per l'output html ho utilizzato **Saxon** sui file **codxml.xml** e **Progetto.xsl** generando l'**output.html** sempre da prompt.

Nell'output html infine ho aggiunto un foglio di stile con css chiamato **stile.css** e un file javascript chiamato **codJS.js** per gestire i bottoni.
